# **App Name**: HuntR

## Core Features:

- Verified Listings: Display only verified property listings based on owner identity, physical visits, and legal checks.
- Smart Search: AI-powered search functionality that uses NLP to understand user intent and find relevant properties, using location data as a tool.
- Category Filters: Allow users to filter properties by type (PG/Hostel, 1 BHK, 2 BHK, Flat/Roommate).
- Owner Contact: Enable direct contact with verified property owners, skipping brokerage fees.
- Property Details: Display detailed information for each property, including photos, amenities, and location.

## Style Guidelines:

- Primary color: Deep trustworthy blue (#0056B3) to inspire confidence and security in property hunting.
- Background color: Very light blue (#E7F1FF), offering a clean, spacious feel.
- Accent color: Vibrant mint green (#00B894) to highlight key interactive elements and call-to-actions.
- Body and headline font: 'Poppins' (sans-serif) for a contemporary, readable style that enhances user experience.
- Use Font Awesome icons with blue or green colors to maintain consistency.
- Cards with modern rounded corners and subtle borders create a friendly design, offering enough visual separation for all properties in view.
- Subtle animations such as hover effects lift cards and change color when clicked
